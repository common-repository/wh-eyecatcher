Adds a floating slogan to eyery page of your website .

== Description ==

Adds a floating slogan to eyery page of your website 

*   "Slogan" is any text you would like to appear as eyecatcher
*   "Style" allows to use some predefined demo styles to play around
*   "CSS" allows editing the CSS definitions to make the eyecatcher slogan appear just as you imagine

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wh-eyecatcher` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->WH Eyecatcher screen to configure the plugin

== Frequently Asked Questions ==

= Can I preview the slogan appearance before using it in the frontend =

Yes, any changes to the slogan text and the CSS definition can be previewed as the backend configuration screen also shows the eyecatcher. The changes you make are immediately affecting the display.
